## Usage 

1) For training simple DNN
```shell
python simple_dnn.py
```

2) For evaluation with
   - untargeted FGSM attacks
   - targeted FGSM attacks
```shell
python fgsm_attacks.py
```

3) For adversarial training with
   - untargeted FGSM attacks 
   - targeted FGSM attacks
```shell
python adversarial_training.py
```
